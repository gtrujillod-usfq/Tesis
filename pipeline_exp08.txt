NODO 2 -- PREPROCESAMIENTO DICOM -> PIL RGB
================================================================================
Archivo        : data_loading.py:169-213 (_load_dicom_as_pil)

Pasos en orden exacto:

  2a. VOI LUT (windowing)
      Operacion : _apply_voi_lut(arr, ds) de pydicom
                  Mapea rango 16-bit al rango de visualizacion usando los
                  parametros de ventana del equipo (clinicamente correcto).
                  Si falla, continua con el array crudo.
      Salida    : array float32 [H, W] en rango de ventana clinica

  2b. Inversion MONOCHROME1
      Operacion : si ds.PhotometricInterpretation == "MONOCHROME1"
                      arr = arr.max() - arr
                  (invierte la escala para que tejido denso quede claro)
      Salida    : array float32 [H, W] en convencion MONOCHROME2

  2c. Normalizacion min-max -> uint8
      Operacion : arr = (arr - arr.min()) / (arr.max() - arr.min()) * 255.0
                  cast a uint8
      Salida    : array uint8 [H, W] con valores [0, 255]

  2d. Conversion a PIL RGB
      Operacion : Image.fromarray(arr).convert("RGB")
                  Replica el canal gris 3 veces.
      Salida    : PIL Image RGB [H, W, 3]

NODO 5 -- CABEZAS DE CLASIFICACION (Dual-Head)

NODO 6 -- POSTPROCESAMIENTO DE CLASIFICACION
================================================================================
Archivo        : main.ipynb seccion 7.4 (funcion generar_informe_para_imagen)
                 models.py:327-368 (predict)

  6a. Softmax
      birads_probs  = F.softmax(outputs['birads'][0],  dim=-1)  -> [5] probs
      density_probs = F.softmax(outputs['density'][0], dim=-1)  -> [4] probs

  6b. ArgMax (clasificacion multiclase pura)
      birads_idx  = int(torch.argmax(birads_probs))   -> entero {0,1,2,3,4}
      density_idx = int(torch.argmax(density_probs))  -> entero {0,1,2,3}
      birads_conf = float(birads_probs[birads_idx])   -> confianza [0,1]

  6c. Score de malignidad (main.ipynb ~linea 1377-1378)
      Formula exacta del codigo:
        malignancy_score = float(birads_probs[3].item() + birads_probs[4].item())
      Semantica: P(BI-RADS 4) + P(BI-RADS 5) = probabilidad acumulada
                 de categorias sospechosas/malignas
      Rango: [0.0, 1.0]

  6d. Threshold Tuning -- Camino B (threshold_tuning.py:56-90)
      Umbral exp08 : 0.120
                     (elegido sobre validacion maximizando indice de Youden;
                      archivo threshold_tuning_camino_b_2026-06-06_04-23-39.json)
      Decision     : si malignancy_score >= 0.120  -> caso SOSPECHOSO/MALIGNO
                     si malignancy_score <  0.120  -> caso BENIGNO
      Metricas en test con umbral 0.120:
        Sensibilidad: 0.7576   FN: 48   Especificidad: 0.7007   FP: 1138

      Produce dos reportes paralelos:
        ArgMax puro : BI-RADS 1-5 multiclase (sensibilidad 0.37, especificidad 0.98)
        Camino B    : benigno vs. sospechoso  (sensibilidad 0.76, especificidad 0.70)

SALIDA DEL NODO:
  birads_idx (0-4), density_idx (0-3), birads_conf (float),
  malignancy_score (float), decision binaria Camino B

  NOTA: el informe usa ArgMax para el nivel BI-RADS 1-5 e incorpora
  malignancy_score como campo adicional. El umbral Camino B es para
  analisis de cribado; no reemplaza el ArgMax en el informe.

NODO 12 -- SALIDA: Informe Estructurado
================================================================================
Archivo        : report_generator.py:381-420 (template / generado por LLM)

Secciones del informe (5, segun ACR BI-RADS Atlas):

  1. TECNICA
     Mamografia digital en proyecciones CC (craneocaudal) y MLO
     (mediolateral oblicua).

  2. COMPOSICION MAMARIA
     Descripcion de densidad segun categoria ACR (A/B/C/D) + nota
     clinica si es C o D (densidad elevada reduce sensibilidad y
     es factor de riesgo independiente).

  3. HALLAZGOS
     Descripcion acorde al nivel BI-RADS, SIN morfologia inventada:
       BR 1-2 -> "no se identifican hallazgos sospechosos de malignidad"
       BR 3   -> "hallazgo probablemente benigno que amerita seguimiento"
       BR 4-5 -> "hallazgo con caracteristicas sospechosas que amerita
                  evaluacion adicional"
     + Score de riesgo de malignidad si se proporciona.

  4. IMPRESION
     Categoria BI-RADS {1-5} con justificacion clinica.

  5. RECOMENDACION
     Conducta clinica ACR estandar para ese nivel BI-RADS.
